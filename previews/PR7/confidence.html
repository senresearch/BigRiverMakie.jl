BigRiverMakie.confidence_plot Function
julia
confidence_plot(x_data, y_labels, error_values; kwargs...)

Generates a confidence interval plot around x_data with error bars as given by error_values. The y-axis is labeled with y_labels.

Arguments

  • x_data: A vector of x values.

  • y_labels: A vector of y labels.

  • error_values: A vector of error values.

  • kwargs: Additional arguments to pass to the confidenceplot! function.

BigRiverMakie.confidenceplot Function
julia
confidenceplot!(x, ϵ; kwargs...)
confidenceplot(x, ϵ; kwargs...)

Generate a confidence interval plot. This is a very simple plot that takes a vector of x values and a vector of error values. The error values are used to create a confidence interval around the x values. If the confidence interval contains 0, then it is considered to be significant and the point is colored dark blue, otherwise it is colored light blue.

This plot is a Makie recipe and does not set any theming options, or even the axis labels. It can of course be customized with the same variety that any Makie plot can be, but for a plot with more default options and the choice to pass in axis labels, see also confidence_plot.

Arguments

  • x: A vector of x values.

  • ϵ: A vector of error values.

Plot type

The plot type alias for the confidenceplot function is ConfidencePlot.

Attributes

errorbarcolor = :black — Color of the error bars. Check the Makie documentation for more color options.

linecolor = :red — Color of the vertical line at 0. Check the Makie documentation for more color options.

linestyle = :dash — Style of the vertical line at 0. Check the Makie documentation for more line style options.

markershape = :rect — Shape of the markers for the scatter plot. Check the Makie documentation for more marker shape options.

markersize = 15 — Size of the markers.

scatterbordercolor = :black — Color of the border of the markers. Check the Makie documentation for more color options.

scatterborderwidth = 1 — Width of the border of the markers.

scattercolors = :Blues_3 — Colors for the markers. This uses the first and last colors of the colormap provided, or if a vector of colors is provided, the first and last colors of the vector. Check the Makie documentation for more color options.

whiskerwidth = 10 — Width of the whiskers of the error bars.
